# 💜 Site Oficial - Dra. Cristiane Bohm dos Santos
**Fisioterapia Forense e Perícia Judicial**

Este repositório contém o código-fonte do site oficial da Dra. Cristiane Bohm dos Santos, hospedado e publicado através do GitHub Pages. O foco do projeto é apresentar os serviços de Consultoria Pericial Documental e Assistência Técnica para advogados e clientes em todo o Brasil.

🌍 **Acesse o site ao vivo:** [crisfisioforense.com.br](https://crisfisioforense.com.br)

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica, acessibilidade e otimização avançada para SEO.
* **CSS3:** Estilização visual, layout responsivo (adaptável para celulares) e variáveis de design.
* **JavaScript (Vanilla):** Interatividade do site, como o menu hambúrguer para mobile, rolagem suave entre as seções e abertura de modais do blog.

## 📁 Estrutura Principal do Projeto
* `index.html`: Arquivo principal contendo todo o conteúdo em texto, links e estrutura SEO.
* `styles.css`: Folha de estilos contendo o design visual, cores e responsividade.
* Imagens (`.png`, `.jpg`): Arquivos de mídia e ícones armazenados na raiz do projeto.
